== Description ==

Set filter by taxonomy.
This plugin set tax filter at admin panel.
If the post type has multiple taxonomy, it appear multiple filter dropdown. :)

This plugin has been managed on Github.(https://github.com/marushu/set_filter_by_taxonomy)
This plugin was created is inspired to JiveDig.(https://gist.github.com/JiveDig)

== Installation ==

This section describes how to install the plugin and get it working.

- Upload `set-filter-by-tax` to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress
- Then it'll be the filter drop-down appears in your custom post type admin column.
